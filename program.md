# データ
## データ型とサイズ
| データ型 | サイズ(byte) | 最大数 |
| :--- | :--- | :--- |
| `char` | 1 | 127 (※符号なし `unsigned` の場合は 255) |
| `unsigned char` | 1 | 255 |
| `short` | 2 | 32,767 |
| `unsigned short` | 2 | 65,535 |
| `int` | 4 | 2,147,483,647 （約21億） |
| `unsigned int` | 4 | 4,294,967,295 （約42億） |
| `long` | 4 または 8 ※1 | 21億 または 約922京 ※1 |
| `unsigned long` | 4 または 8 ※1 | 42億 または 約1844京 ※1 |
| `long long` | 8 | 9,223,372,036,854,775,807 （約922京） |
| `unsigned long long` | 8 | 18,446,744,073,709,551,615 （約1844京） |
| `float` | 4 | 約 3.402823e+38 |
| `double` | 8 | 約 1.797693e+308 |
| `long double` | 8, 12 または 16 ※2 | 環境に依存（少なくとも double 以上） |
## 文字
### サイズ
| 文字の種類 | UTF-8 | Shift_JIS |
| :--- | :--- | :--- |
| **半角英数字** (`A`, `1`) | 1 byte | 1 byte |
| **半角カナ** (`ｱ`) | 3 byte | 1 byte |
| **全角ひらがな・カタカナ** (`あ`) | 3 byte | 2 byte |
| **一般的な漢字** (`漢`) | 3 byte | 2 byte |
| **絵文字・サロゲートペア** (`🍣`) | 4 byte | (対応不可/化ける) |
### コード
- 自己同期性がある('あ'の中に'a'の羅列が含まれる、このようなことが起きなくなっている)
| バイト数 | 1バイト目の先頭ビット | 2バイト目以降 | 役割・主な用途 |
| :--- | :--- | :--- | :--- |
| **1 byte** | `0xxxxxxx` | - | ASCII (英数字・記号) |
| **2 bytes** | `110xxxxx` | `10xxxxxx` | ギリシャ文字、アラビア文字等 |
| **3 bytes** | `1110xxxx` | `10xxxxxx` (×2) | 日本語（ひらがな・漢字等） |
| **4 bytes** | `11110xxx` | `10xxxxxx` (×3) | 絵文字、稀少な漢字等 |
| **(継続用)** | `10xxxxxx` | - | マルチバイトの2〜4手目専用 |

# 定義
### 演算子
- 優先順位を持つ
- 言語の構成要素(コンパイル時に値を確定させる、アドレスに処理を引く必要性が低い)
- 記法の自由度が保証
