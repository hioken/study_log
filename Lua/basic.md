# 概要
## 概要
- ホストプログラムをサポートする前提
- 依存先がCのみで、Cのコンパイラさえあればどんな環境でも(たとえOSがなくとも)動く
## メモリ管理
- Luaはメモリを確保しない
- ホストが使っているメモリ領域を、Luaに貸し出す
- ホストへの領域返却はLua側で行う(LuaのGC)
## 共通スタック
- ホスト側がLuaが直接読めるような関数を用意する
- あらかじめLuaに読み取ってほしい領域を指定して、読み取ってほしい順にスタックさせていく
- Luaは必ず決まった順番で実行して、決まった順番に結果を返す
- ホストはLuaが処理を終えるまで待機する

# データ
## テーブル
- Lua唯一の構造体
- 配列パートとハッシュパートを持っている
  - 連番のキーの場合は配列パート、それ以外はハッシュパートで管理する
- それぞれメモリ管理が異なり、ハッシュパートは通常の管理、配列パートではC言語のような管理をする

## メタテーブル
- テーブルに対して未知の処理をした時の挙動を設定できるもの

| メタメソッド名 | 引数 | それぞれの引数の使用 | 説明 |
| :--- | :--- | :--- | :--- |
| `__index` | `(t, k)` | `t`: 対象テーブル, `k`: 存在しなかったキー | 存在しないキーを参照した時に呼ばれる。関数または代替テーブルを指定可能。 |
| `__newindex` | `(t, k, v)` | `t`: 対象テーブル, `k`: キー, `v`: 代入しようとした値 | 存在しないキーへ値を代入しようとした時に呼ばれる。 |
| `__call` | `(t, ...)` | `t`: 関数として呼ばれたテーブル, `...`: 渡された引数 | テーブルを `t()` のように関数として呼び出した時の挙動を定義。 |
| `__tostring` | `(t)` | `t`: 対象テーブル | `tostring(t)` や `print(t)` を実行した際の文字列表現を定義。 |
| `__len` | `(t)` | `t`: 対象テーブル | `#t`（長さ演算子）を実行した時の戻り値を定義。 |
| `__metatable` | (なし) | (メタテーブルに値を設定する) | `getmetatable` の戻り値を固定し、メタテーブル自体を隠蔽・保護する。 |
| `__add` | `(a, b)` | `a`, `b`: 加算の左右の項 | `+` 演算子の挙動を定義。 |
| `__sub` | `(a, b)` | `a`, `b`: 減算の左右の項 | `-` 演算子の挙動を定義。 |
| `__mul` | `(a, b)` | `a`, `b`: 乗算の左右の項 | `*` 演算子の挙動を定義。 |
| `__div` | `(a, b)` | `a`, `b`: 除算の左右の項 | `/` 演算子の挙動を定義。 |
| `__idiv` | `(a, b)` | `a`, `b`: 整数除算の左右の項 | `//` 演算子の挙動を定義。 |
| `__mod` | `(a, b)` | `a`, `b`: 剰余の左右の項 | `%` 演算子の挙動を定義。 |
| `__pow` | `(a, b)` | `a`, `b`: 累乗の左右の項 | `^` 演算子の挙動を定義。 |
| `__unm` | `(a)` | `a`: 対象の値 | `-a`（単項マイナス）の挙動を定義。 |
| `__band` | `(a, b)` | `a`, `b`: ビットANDの左右の項 | `&` 演算子（ビット論理積）の挙動を定義。 |
| `__bor` | `(a, b)` | `a`, `b`: ビットORの左右の項 | `\|` 演算子（ビット論理和）の挙動を定義。 |
| `__bxor` | `(a, b)` | `a`, `b`: ビットXORの左右の項 | `~` 演算子（ビット排他的論理和）の挙動を定義。 |
| `__bnot` | `(a)` | `a`: 対象の値 | `~a`（ビット反転）の挙動を定義。 |
| `__shl` | `(a, b)` | `a`: 値, `b`: シフト量 | `<<` 演算子（左シフト）の挙動を定義。 |
| `__shr` | `(a, b)` | `a`: 値, `b`: シフト量 | `>>` 演算子（右シフト）の挙動を定義。 |
| `__concat` | `(a, b)` | `a`, `b`: 結合する左右の項 | `..` 演算子（文字列結合）の挙動を定義。 |
| `__eq` | `(a, b)` | `a`, `b`: 比較する左右の項 | `==` 演算子の挙動を定義。 |
| `__lt` | `(a, b)` | `a`, `b`: 比較する左右の項 | `<` 演算子の挙動を定義。 |
| `__le` | `(a, b)` | `a`, `b`: 比較する左右の項 | `<=` 演算子の挙動を定義。 |
| `__gc` | `(t)` | `t`: 回収される対象 | オブジェクトがガベージコレクションで回収される直前に呼ばれる。 |
| `__close` | `(t, err)` | `t`: 閉じる対象, `err`: エラーオブジェクト | `to-be-closed` 変数がスコープを抜ける際に呼ばれる (Lua 5.4新機能)。 |
| `__name` | (なし) | (文字列を設定する) | `type()` やエラーメッセージで表示される型名を定義する。 |

```Lua
local my_behavior = {
  __index = function(t, key)
    return "そんなキーはありません: " .. key
  end
}

local t = {}
setmetatable(t, my_behavior)

print(t.hoge) -- "そんなキーはありません: hoge" と表示される
```
